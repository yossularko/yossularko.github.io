Here's a little background:

Inilah sedikit tentang saya, sebelum ke dunia programming saya adalah seorang Graphic Designer dengan pengalaman kurang lebih 5 tahun, Video Editor pengalaman kurang lebih 2 tahun, dan terakhir IT Support pengalaman kurang lebih 1 tahun. Kini saya beralih ke dunia programming, motivasi awal saya adalah ingin upgrade skill programming karena waktu itu saya hanya mempunyai kemampuan html, css, dan php dasar saja (ilmu dasar ini saya dapat ketika di bangku kuliah). Dengan berbekal googling dan tekun mencari tutorial gratis di youtube saya berhasil membuat sebuah web portfolio dengan html, css, dan javascript. Pada saat belajar otodidak ada masa dimana saya merasa bangga dalam memecahkan masalah, error dan menambah fitur baru. Hingga timbul rasa penasaran untuk mengulik lebih luas ilmu pemrograman. Dalam belajar pemrograman saya fokuskan pada sisi FrontEnd hingga saya mendapat project pertama membuat sebuah website Company Profile sekitar 2 tahun yang lalu menggunakan WordPress, dan kini sudah mengerjakan beberapa project dengan menggunakan React JS, Next JS, dan React Native. Dan project-project baru sekarang sudah menggunakan TypeScript.
